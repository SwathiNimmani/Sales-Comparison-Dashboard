# Sales Comparison Dashboard

## Project Overview

This project demonstrates an interactive Tableau dashboard developed to compare sales performance between two regions using the Sample Superstore dataset.

## Objective

To build an interactive dashboard that enables users to compare regional sales performance, customer behaviour, and key business metrics to support data-driven decision-making.

## Tools Used

- Tableau
- Sample Superstore Dataset
- Parameters
- Calculated Fields
- Hierarchies

## Features

- Regional comparison
- Sales KPIs
- Customer analysis
- Order analysis
- Interactive dashboard

## Outcome

The dashboard provides a clear comparison of regional performance through interactive visualisations, helping users identify trends and make informed business decisions.
